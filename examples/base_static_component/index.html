该 Demo 展示了如何使用 TalqsTemplate 添加自定义组件

在下面的例子中，我们将自定义两个组件，并且添加到解析模板中

首选，我们可以通过两种方式定义组件

第一种： 直接在 HTML 文件中定义，例如：

第二种： 在 js 文件中定义组件字符串

其次，需要通过 registerComponent 方法来注册定义好的组件

在该 Demo 中，我们定义了两个组件，分别为标签组件和机构组件，下面我们通过 API 将其添加到组件集合中

        
// 注册组件
// 通过HTML注册的组件需要获取innerHTML
var questionLabel = document.getElementById('questionLabel').innerHTML;
TalqsTemplate.registerComponent({
  questionLabel,
  questionOrganization
})

最后，通过updateTemplateList更新解区域的模板结构
这里需要注意的是对于 questionOrganization 试题机构组件，我们更新时指定了其出现的顺序 index.
在解析模板中试题机构将在组件列表中的第四个呈现
如果不指定 index，默认将组件添加到列表的末尾 

        
TalqsTemplate.updateTemplateList('analyzeWrapper', [
  {
    component: 'questionLabel'
  },
  { 
    component: 'questionOrganization',
    index: 3
  }
])
